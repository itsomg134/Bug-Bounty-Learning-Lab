Responsible Disclosure Steps

  1. Identify vulnerability on an authorized program (HackerOne, Bugcrowd, etc.)
  2. Document steps to reproduce
  3. Report privately to the company via their security contact
  4. Never exploit beyond proof of concept
  5. Wait for disclosure timeline agreement

Common Vulnerability Types (OWASP Top 10)

  • SQL Injection (SQLi)
  • Cross-Site Scripting (XSS)
  • Cross-Site Request Forgery (CSRF)
  • Server-Side Request Forgery (SSRF)
  • Broken Access Control
